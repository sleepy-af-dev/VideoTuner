# Code of Conduct

Be kind to one another. Treat people with respect, assume good faith, and keep discussion focused on the project.

## Not welcome

Harassment, discrimination, personal attacks, and sharing others' private information.

## Enforcement

This project is maintained by sleepy-af-dev, who may edit or remove contributions and block anyone whose behavior is harmful.

## Reporting

If someone is behaving badly, report it privately using GitHub's [report abuse or spam](https://docs.github.com/en/communities/maintaining-your-safety-on-github/reporting-abuse-or-spam) tool. Reports are handled confidentially.
